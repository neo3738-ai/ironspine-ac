# IronSpine AC

AI-assisted development environment with conversation analysis, task tracking, and MCP integration for AI coding workflows.

## Features

- **Conversation analysis** — read and analyze Claude conversation history
- **Desktop UI** — manage your AI development sessions from a clean interface
- **MCP server** (`ironspine-ac-mcp`) — AI assistant integration for reading sessions and context
- **Local storage** — all data stays on your machine

## Binaries

Ships two executables:
- `ironspine-ac` (`ironspine_ac_ui.exe`) — the desktop application
- `ironspine-ac-mcp` (`ironspine_ac_mcp.exe`) — the MCP server
